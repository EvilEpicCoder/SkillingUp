# Hexo - How to create new page

## Basics
### 3 tipes of Pages:
* Post
 * Post is like news
 `hexo new Hello world!`
* Draft
 * Draft unpublished news
 `hexo new draft Second hidden hello world!`
 * Make it published:
 `hexo publish Second hidden hello world!`
* Pages
 * Menu like pages
 `hexo new page About`
